## Information We Collect

### 1. Ticket Data
The App accesses and processes the following information from Freshdesk tickets:
- Ticket subject
- Ticket description
- Any additional contextual information provided by the user

### 2. Configuration Data
- OpenAI API Key (stored securely in Freshdesk installation parameters)
- System prompts and configuration settings
- Debug settings (when enabled)

### 3. API Communication Data
- Requests sent to OpenAI API
- Responses received from OpenAI API

## How We Use Your Information

The information collected is used exclusively for the following purposes:

1. **Ticket Analysis**: To process ticket content and generate AI-powered responses using OpenAI's ChatGPT service
2. **Service Functionality**: To enable the core features of the App within Freshdesk
3. **Response Generation**: To create contextual, professional support responses
4. **Debugging**: When debug mode is enabled, to maintain logs for troubleshooting purposes

## Data Storage and Retention

### Storage Locations
- **Ticket Data**: Processed in real-time and not permanently stored by the App
- **API Keys and Configuration**: Stored securely within Freshdesk's installation parameters system
- **Logs**: When debug mode is enabled, logs are stored temporarily in the browser console

### Data Retention
- The App does not permanently store ticket content or AI-generated responses
- Configuration data (API keys, prompts) is retained as long as the App is installed
- Upon app uninstallation, all configuration data stored by the App is automatically deleted from Freshdesk
- Any data sent to OpenAI is subject to [OpenAI's data retention policies](https://openai.com/policies/privacy-policy)

## Third-Party Services

### OpenAI Integration
This App integrates with OpenAI's API to provide AI-powered ticket analysis and response generation. When you use this App:

- Ticket content (subject and description) is sent to OpenAI's servers for processing
- OpenAI processes this data according to their [Privacy Policy](https://openai.com/policies/privacy-policy) and [API Data Usage Policies](https://openai.com/policies/api-data-usage-policies)
- **Location**: OpenAI data centers (primarily United States)
- **Purpose**: AI-powered text generation and analysis
- **Data Shared**: Ticket subject, description, and any additional context provided by the user

We recommend reviewing OpenAI's data usage policies to understand how they handle your data.

## Data Security

We implement the following security measures:

1. **Encryption in Transit**: All communications with OpenAI API use HTTPS encryption (TLS/SSL)
2. **Secure Storage**: API keys are stored using Freshdesk's secure installation parameters system
3. **No Permanent Storage**: Ticket data is not permanently stored by the App
4. **Access Control**: Only authorized users with access to your Freshdesk instance can use the App
5. **No Data Export**: The App does not provide features to export or download user data

## Data Access and Logging

- When debug mode is **disabled** (default): Minimal logging is performed
- When debug mode is **enabled**: Detailed logs are written to the browser console for troubleshooting purposes
- Logs may contain ticket subjects and descriptions for debugging purposes
- All logs are temporary and stored only in the browser session
